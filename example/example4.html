Style on dragging

In the examples here you have seen it already: while dragging the DIV changes transparency, gets a border and a shadow.
What you need to do is to define a css class .isdragging and apply wanted styles here.

Snippet

            /* define styling on dragging */
            .isdragging{
                opacity: 0.6;
                border: 2px dashed;
                box-shadow: 0.2em 0.2em 1em rgba(0,0,0,0.4);
            }
