Welcome

Welcome to the example section of the ADDI class!
With ADDI you can simply add a drag and drop functionality for DIVs (or other html tags) in a webpage.

You need...

  • html code: create a tag with class "draggable" ... and maybe other classes for styling
  • javascript: load the addi.js in your page and run its init method

Try it

Snippet

This is the needed code snippet for it: 

            <div class="draggable box1">
                Drag me around on page :-)<br>
                Here could be some more content.
            </div>

            ... 
            
            <!-- at the end of your document -->

            <script src="/javascript/addi.js">
            <script>
                addi.init();
            </script>
