On screen or on page

The DIVs can be moved relative to the page or relative to the screen.
Both DIVs below are movable. If you scroll the page then one is fixed on the screen and theother one stays on its position relative to the page content.

Snippet

What the trick is? You can influence it by the css attribute "position".

  • absolute - make it movable on the page
  • fixed - make it movable on the screen
            <div class="draggable box2">
                Drag me around on screen :-)
            </div>

            <div class="draggable box1">
                Drag me around on page :-)<br>
                Here could be some more content.
            </div>

The CSS code: 

            .box1{
                ...
                position: absolute;
            }
            .box2{
                ...
                position: fixed;
            }

Suggestion: Create a more readable css class

Add css classes to describe the behaviour in a more readable way.: 

        .draggable-onpage{position: absolute;}
        .draggable-onscreen{position: fixed;}

To write in your html code 

            <div class="draggable draggable-onscreen box2">
                Drag me around on screen :-)
            </div>

            <div class="draggable draggable-onpage box1">
                Drag me around on page :-)<br>
                Here could be some more content.
            </div>
