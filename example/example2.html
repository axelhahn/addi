Get started Custom class On screen or on page Style on dragging Reset position

Use a given class

If you have a class for draggable elements already ...
then call the init method with the name of your wanted css class.

In the example below I change it from the default "draggable" to "dragclass". 

            <div class="dragclass box1">
                Drag me around :-)
            </div>

            ... 
            
            <!-- at the end of your document -->

            <script src="/javascript/addi.js">
            <script>
                addi.init("dragclass");
            </script>
